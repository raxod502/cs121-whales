About W.H.A.L.E.S.

W.H.A.L.E.S. is a web application that allows you to play chess against an AI opponent using neural networks and deep learning.

Usage

  • From the home page, you can select a color for your pieces and an AI model to play against, then start a game.
  • When it is your turn, click and drag one of your pieces to make a move. After thinking, the AI will respond and return control to you.
  • At any time, you can also undo the last move, restart the game from the beginning, or return to the home page to change your selected color or AI model.
  • To save a game, simply bookmark the page. If you return later, you can start right where you left off.

About the AI

We would like to give a special thanks to the contributors of chess-alpha-zero, whose pretrained neural network was used for the intermediate and hard AI difficulties. Their network has been trained to output a (policy, value) prediction for a chess game state input. The policy is an array of probabilities of representing the likelihood of making each legal move. The value represents the likelihood of the moving player winning the game.

Below is a quick summary of how each difficulty level selects a move.

  • Easy: Chooses a random move from a list of legal moves
  • Intermediate: Utilizes a minimax algorithm of depth 1, using the value of the board from the neural network as the evaluation function
  • Hard: Utilizes a minimax algorithm depth 2, using the value of the board from the neural network as the evaluation function
